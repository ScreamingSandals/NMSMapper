Using the mappings with NMSMapper (Gradle only)
Add the repository to your settings.gradle: 
                        
pluginManagement {
    repositories {
        maven {
            url = "https://repo.screamingsandals.org/public/"
        }

        gradlePluginPortal()
    }
}

rootProject.name = 'YourProject'
Configuring the plugin: 
                    
plugins {
	// Other plugins
	id 'org.screamingsandals.nms-mapper'
}


/* ... Other stuff ... */

sourceSets.main.java.srcDirs = ['src/generated/java', 'src/main/java']

nmsGen {
	basePackage = "com.example.nms.accessors"
	sourceSet = "src/generated/java"
	cleanOnRebuild = true

	reqClass('[[${accessorRequireClass}]]')
	/* ... Other classes ... */
}

tasks.getByName('compileJava').dependsOn('generateNmsComponents')
Using the accessor: 
                        
import com.example.nms.accessors.[[${accessorName}]];

// ...

Class<?> theClass = [[${accessorName}]].getType();
